Bootstrap Structure and Files

CSS Folder

The CSS folder contains Bootstrap stylesheet files used for designing layouts, buttons, colors, typography, spacing, and responsive webpages.

JS Folder

The JS folder contains JavaScript files used for interactive components like modals, dropdowns, carousels, accordions, and navbar toggles.

Icons Folder

The Icons folder contains Bootstrap icons used to add graphical symbols such as home, search, user, and social media icons.
